About

A lightweight platform that stays learning focused

What this site is

A free, public learning hub for quantum annealing, QUBO modeling, and routing applications. Videos are hosted on YouTube. Code and documents live in GitHub.

Goal: help you start quickly, then learn by building.

How to maintain

  1. Edit data/courses.json and replace YouTube IDs
  2. Add resource links for slides, notebooks, and papers
  3. Commit and push, GitHub Pages updates after deployment
