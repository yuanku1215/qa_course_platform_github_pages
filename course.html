Problem 1
Variables

A system has a binary switch y and a production level x. If the system is off, production is zero. If it is on, production must be between 10 and 50.

Answer 
y ∈ {0,1}
10y ≤ x ≤ 50y

The binary variable activates the feasible range of x. When y = 0, x is forced to zero automatically.

Problem 2
Constraints

A commuter must choose exactly one transportation mode: public transit or driving. 

x₁ + x₂ = 1
x₁, x₂ ∈ {0,1}

Exactly one choice is enforced by an equality constraint.
