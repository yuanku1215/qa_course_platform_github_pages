Problem 1 · Variables

Question
A system has an on off switch y and a production level x. If off, x = 0. If on, x ∈ [10, 50].
Answer
y ∈ {0,1}
10y ≤ x ≤ 50y
Explanation
Binary y activates the feasible range of x. When y = 0, x is forced to zero.
